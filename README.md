## 🎾 Об игре

**NowHitTheBallVR** — это симулятор настольного тенниса для VR, который стирает грань между виртуальным и реальным миром. В отличие от обычных VR-игр, где вы управляете ракеткой через контроллеры, здесь вы используете **реальную ракетку** с закреплённым трекером HTC Vive.

Проект разработан на **Unity** с использованием **SteamVR SDK** и предлагает уникальный опыт игры, максимально приближенный к настоящему теннису.

---

## ✨ Ключевые особенности

- 🏓 **Реальная ракетка** — трекер HTC Vive прикрепляется к вашей собственной ракетке, обеспечивая полный контроль над каждым ударом.
- 🎯 **Калибровка** — специальная система калибровки позволяет идеально совместить виртуальную модель ракетки с её реальным положением в пространстве.
- 🌍 **Разные карты** — играйте в классическом спортивном зале или на других уникальных аренах.
- 🎯 **Режимы с мишенями** — тренируйте точность, попадая по мишеням через стол в специальных режимах.
- 🔄 **Физика** — реалистичная физика полёта мяча и отскоков, адаптированная под VR-взаимодействие.

## 🚀 Установка и запуск

### Требования
- **ОС:** Windows 10/11
- **VR-шлем:** HTC Vive / Valve Index / любой SteamVR-совместимый шлем
- **Трекер:** HTC Vive Tracker (3.0)
- **Ракетка:** Любая настольная ракетка с креплением для трекера
- **Unity:** 2022.3 LTS или новее (для разработки)

### Сборка из исходников
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш-аккаунт/NowHitTheBallVR.git
   ```
2. Откройте проект в Unity Hub (выберите версию 2022.3+).
3. Убедитесь, что в Package Manager установлен **SteamVR Plugin**.
4. Настройте параметры сборки:
   - `File → Build Settings`
   - Платформа: **Windows, Mac, Linux**
   - Архитектура: **x86_64**
   - Настройки VR: включите `Virtual Reality Supported`
5. Соберите проект (`Build`) или запустите в редакторе с подключённым VR-оборудованием.
